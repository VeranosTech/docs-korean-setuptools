Setuptools 문서에 온 것을 환영한다!
=====================================

Setuptools는 아래를 포함하는 파이썬 프로젝트 패키징을 편리하게 만들기 위해 디자인 된 안정적인 라이브러리로서 완벽한 기능을 갖추고 있고 활동적으로 유지관리되고 있다:

 - 파이썬 패키지와 모듈 정의
 - 배포 패키지 메타 데이터
 - 테스트 훅
 - 프로젝트 설치
 - 플랫폼 별 세부 사항
 - 파이썬 3 지원
