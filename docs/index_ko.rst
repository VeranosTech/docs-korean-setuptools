Setuptools 문서
=====================================

Setuptools는 파이썬 프로젝트 패키징을 편리하게 만들기 위해 설계된 라이브러리이다.
Setuptools는 이에 필요한 다음과 같은 기능을 갖추고 있으며 안정적으로 관리되고 있다.

 - 파이썬 패키지와 모듈 정의
 - 배포 패키지 메타 데이터
 - 테스트 훅
 - 프로젝트 설치
 - 플랫폼 별 세부 사항
 - 파이썬 3 지원
